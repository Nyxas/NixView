Welcome to the GitHub pages of the Nix View project

NixView is a viewer for nix data files. It is rather experimental and under development. The viewer provides an overview over the data stored in the file and allows to plot 1D data.

The test folder contains a sample hdf5 - nix file that contains several kinds of data and shows some of the functions supported by the nix data model like the tagging of segments or regions.
